空气炸锅菜谱

烤红薯

160℃ x 45min

半成品鸡块

200℃ x 10min

烤鸡翅根

  1. 刀背拍打后腌制几个小时（一夜也可以），配料如下
    • 十三香
    • 蒜末
    • 盐
    • 植物油
  2. 160℃ x 10min

花菜

  1. 洗净切好后，放入盐，油，搅拌均匀
  2. 160℃ x 10min，注意不时查看，避免焦糊

烤鸡腿排

  1. 猪排/去骨鸡腿排洗净后放入烧烤酱，盐，油，胡椒粉，辣椒粉，腌制一晚
  2. 准备蛋液，面粉，先黏上蛋液后裹上淀粉,或者面包糠
  3. 160℃ x 15min,隔五分钟翻个面

盐烤鸡腿排

  1. 去骨鸡腿排洗净后放入盐，腌制一晚
  2. 电饼铛煎熟即可

面包糠

  1. 吐司撕成块, 喷油后, 180℃ x 10min

炒菜

蛋炒饭1

  1. 加油，一半葱花（大葱切碎也可）稍微炒下后打入鸡蛋
  2. 鸡蛋快炒至固体时加入米饭（米饭上可以裹上一些蛋液），翻炒同时把米饭块弄散（米饭放入冰箱之前最好戳散，不然炒的时候很费力）
  3. 米饭翻炒一会后，加入生抽，盐炒匀，再加入另一半葱花，最后加入味精翻炒几下即可

蛋炒饭2

  1. 鸡蛋打散, 加油炒熟后盛出
  2. 加油, 葱花炒香后加入加入米饭，炒散, 可以再次加入葱花
  3. 米饭翻炒一会后，加入炒好的鸡蛋，再次翻炒, 可以再加入一些葱花
  4. 米饭翻炒一会后，加入生抽炒匀，再加入一些葱花，最后加入味精翻炒几下即可

老干妈炒饭

  1. 鸡蛋打散, 加油炒熟后盛出
  2. 加油, 加入一勺老干妈小火炒香后, 加入米饭炒散
  3. 米饭翻炒一会后，加入炒好的鸡蛋，再次翻炒
  4. 米饭翻炒一会后，加入少量生抽炒匀，最后加入味精翻炒几下即可

青椒炒蛋

  1. 青椒切丝，放入锅中中火炒至断生
  2. 调小火，鸡蛋打到锅中青椒上炒熟，炒匀后加入盐，味精即可

青椒炒肉

  1. 猪肉切丝，青椒切丝
  2. 猪肉放入锅中中火稍微炒下, 加入生抽再稍微翻炒
  3. 青椒放入炒至断生, 加入盐炒匀后，加入味精即可

青椒炒鸡块

  1. 锅中放油，加大葱末，姜末，蒜末，花椒，八角，炒香
  2. 鸡块放入锅中中火炒熟, 加入生抽，老抽再稍微翻炒
  3. 青椒丝放入炒至断生, 加入盐炒匀后，加入味精即可

虾仁炒蛋

  1. 虾仁焯水后，放入碗中，加入鸡蛋，葱花，盐，搅匀，炒熟即可

番茄鸡蛋汤

  1. 番茄切丁，切葱花，鸡蛋打散
  2. 加一半葱花爆香，加番茄炒至糊状
  3. 加水烧开后，一边打入鸡蛋液，一边搅拌（不要太快，不然鸡蛋会很散）
  4. 关火后，加入剩余葱花，适量生抽，香油，盐即可

煎猪排，鸡腿排

  1. 猪排/去骨鸡腿排洗净后放入烧烤酱，盐，油，胡椒粉，辣椒粉，腌制一晚
  2. 准备蛋液，面粉，先黏上蛋液后裹上淀粉
  3. 平底锅放油铺满底部，加热到160度后，放入猪排，2分钟后翻面，再煎2分钟
  4. 温度调至120度后，间隔一会翻面煎至熟透即可，4分钟左右吧，怕不熟可以适当延长时间，不炸糊即可

洋葱炒鸡蛋

  1. 鸡蛋炒成块状后盛出备用
  2. 加入洋葱翻炒几分钟后, 加入鸡蛋继续翻炒
  3. 加入生抽, 盐翻炒, 停火后加入鸡精翻炒后出锅

蒜苗炒鸡蛋

做法同上，想要鸡蛋嫩一点可以往鸡蛋液里加点水搅拌均匀

炒土豆丝

  1. 土豆切丝, 甩干水分
  2. 加葱姜蒜爆香后, 加入土豆丝翻炒, 防止粘锅, 需要时常翻炒, 可以翻锅和炒勺结合(用电磁炉只要前后抽拉就行了)
  3. 加入生抽, 盐翻炒, 停火后加入鸡精翻炒后出锅

辣椒炒鸡胗

  1. 鸡胗洗干净, 切丝, 青椒也切条
  2. 加油, 鸡胗炒至变色后盛出
  3. 加油, 加入干辣椒, 蒜爆香, 再加入鸡胗翻炒
  4. 加入青椒翻炒后, 再加入生抽, 蚝油, 青椒熟了即可出锅

辣炒腊肠

  1. 腊肠斜切片, 辣椒切丝, 可以适当保留一些辣椒籽
  2. 加油(适当少一些, 腊肠也出油), 加入腊肠翻炒, 炒至卷曲后放入辣椒
  3. 继续翻炒, 加入盐, 酱油, 最后加入鸡精即可

炒花菜

  1. 花菜切小块, 准备葱蒜切末
  2. 加油, (可搭配五花肉)加入葱蒜爆香后, 加入花菜翻炒
  3. 加入盐, 酱油, 适当翻炒
  4. 加入少量热水, 盖上锅盖焖, 中间适当翻炒, 直至花菜熟

西兰花炒鸡肉

  1. 西兰花切小块焯水, 鸡肉焯水后撕成条, 准备葱姜蒜切末
  2. 加油, 加入鸡肉翻炒, 再加入盐,蚝油, 再加入葱姜蒜翻炒后, 加入西兰花翻炒
  3. 加入盐, 酱油, 适当翻炒, 最后加入味精即可

香菇炒肉

  1. 香菇切片焯水(直接炒菜会发酸), 猪肉切条, 准备葱姜蒜切末
  2. 加油, 加入猪肉翻炒, 再加入盐,蚝油, 再加入葱姜蒜翻炒后, 加入香菇翻炒
  3. 加入盐, 酱油, 适当翻炒, 最后加入味精即可

西红柿炒蛋

  1. 鸡蛋液加少量水, 盐, 搅拌均匀, 煎好后盛出
  2. 西红柿表面划浅十字, 开水从顶部浇下, 持续10秒钟后, 去皮切条
  3. 加油, 放入西红柿煎, 加入糖, 盐, 煎至糊状后, 放入鸡蛋, 搅拌均匀后即可
  4. 最后加入葱花

回锅肉

  1. 五花肉焯水后切薄片，蒜苗叶子和茎分开切段
  2. 加油，下入五花肉片翻炒一会后，加入老抽少许，盐，翻炒后再加入蒜苗茎段
  3. 翻炒一会后再加入蒜苗叶翻炒，差不多后加入盐，生抽，最后加入味精

日式牛肉盖浇饭

  1. 锅中加水，加适量生抽，白糖，盐后煮开
  2. 锅中加入切成长条的洋葱，中小火煮软
  3. 肥牛卷过水后放入锅中，小火煮，边煮边搅拌，并漂去浮沫
  4. 加入姜末，小火再煮五分钟
  5. 盛出放到米饭上即可

午餐肉

  1. 鸡胸肉加葱末打成肉泥
  2. 将淀粉,面粉,鸡蛋,水调成稀面糊后, 放入肉末搅拌
  3. 再加入糖, 鸡精, 姜末, 白胡椒粉, 料酒, 食用油后搅拌均匀
  4. 倒入玻璃饭盒中, 铝箔纸盖好, 放入上锅蒸30min

炒粉/炒面

  1. 粉丝/面煮熟，鸡蛋炒熟， 卷心菜切丝备用
  2. 葱末爆香，放入卷心菜翻炒
  3. 放入鸡蛋翻炒，再放入面翻炒
  4. 加酱油翻炒，加味精出锅

葱油拌面

  1. 细面上锅蒸15min，之后打散，可以保存之后吃
  2. 小葱切段后放入油锅，小火煎至棕色，捞出葱段，加入生抽，少量老抽，糖后再小火加热拌匀
  3. 正好的面条滚水中煮2min
  4. 面条捞出，放入葱油和葱段搅匀

萝卜炖肋排

  1. 肋排冷水下锅焯水, 萝卜切片
  2. 八角, 香叶, 花椒, 辣椒, 炒香, 下肋排, 放蚝油, 生抽, 老抽, 盐, 适当翻炒
  3. 加水(开水最好), 加萝卜, 再加生抽, 盐, 炖40min, 适当去浮沫

电饭锅炖肋排

  1. 肋排冷水下锅焯水, 萝卜切片
  2. 肋排放入电饭锅, 加水, 萝卜, 再放入买的成品卤料, 再加生抽, 盐
  3. 煮60min出锅

炖萝卜

  1. 萝卜去皮, 切成4cm的圆柱块, 在其中一面上切2cm深的十字花刀方便煮熟入味
  2. 萝卜放入电饭锅, 加入淘米水(没有的话, 凉水+加点米也可以), 炖30min
  3. 倒掉水后重新加凉水, 加两勺生抽+鸡精(鸡精可以替换为一个鸡腿或者等量猪肉+料酒), 炖40min

凉拌腐竹木耳

  1. 腐竹泡发(40度温水加盐20min), 木耳泡发
  2. 烧开水, 腐竹木耳焯水
  3. 葱末姜末老干妈芝麻放入碗中, 烧热油浇上后加一大勺生抽, 再加鸡精, 糖搅拌
  4. 酱料淋到腐竹木耳上后搅拌即可

咖喱饭

  1. 准备洋葱切条, 鸡肉切块(小苏打适当腌制), 土豆切块, 胡萝卜切块
  2. 加油一次下洋葱, 鸡肉, 土豆, 胡萝卜翻炒后, 加水炖煮至胡萝卜土豆软烂, 中途漂浮沫
  3. 加入买的咖喱, 完全融化后再适当炖煮即可

蒸肉

  1. 准备猪肉切块, 红薯或土豆切块,
  2. 准备酱汁, 生抽, 老抽, 植物油, 蚝油, 搅拌均匀
  3. 把肉放入蒸肉粉中搅拌后, 加入酱汁, 混合均匀, 腌制十分钟
  4. 把红薯或土豆铺到盘底, 放上肉, 开水上锅蒸30min即可

盐水鸭

  1. 锅中加入花椒, 八角, 盐, 小火翻炒至微黄后放凉
  2. 把调料涂到洗干净的鸭腿上, 保鲜膜包好腌制一晚
  3. 锅中放入鸭腿, 八角, 生姜, 冷水下锅, 水开后小火煮30min即可

韭菜炒猪肝

  1. 韭菜切段
  2. 猪肝洗干净后切片，加生抽，料酒，白糖，胡椒粉搅拌，再加入淀粉腌制一会
  3. 锅中加油，葱姜爆香，导入猪肝翻炒变色后盛出
  4. 锅中加油，加入韭菜翻炒，加盐调味，再加入猪肝翻炒后即可

卤鸡腿/鸭腿

  1. 肉洗干净后, 放入开水锅中汆5分钟
  2. 肉放入电饭锅, 加水没过肉
  3. 锅里加入卤料(沃尔玛惠宜炖卤料:桂皮、小茴香、芫荽、八角、橘皮、甘草、白芷、高良姜、月桂叶(香叶)、香茅), 花椒, 大葱, 两勺生抽, 一勺老抽, 蚝油或者鸡精, 一勺盐, 两勺糖
  4. 开电饭锅炖30min(鸭腿40min)

卤牛肉

  1. 牛肉放入冷水中浸泡1h,去血水
  2. 凉水下锅焯水牛肉20min, 漂浮沫
  3. 锅里加入桂皮, 月桂叶(香叶), 花椒, 大葱, 姜, 两勺生抽, 一勺老抽, 一勺盐, 两勺糖
  4. 电饭锅炖1h后,保温一夜

卤蛋

  1. 鸡蛋煮熟后外壳敲裂放入卤水中煮30min后关火留在卤水中过夜

鸡丝凉面

  1. 鸡胸肉煮熟撕成丝, 黄瓜拍扁切丝
  2. 面煮熟过凉水
  3. 小葱末, 蒜末, 老干妈, 芝麻放到碗里后淋热油
  4. 之后再加两勺生抽, 一勺醋, 蚝油或者鸡精, 适量盐, 一勺白糖后搅匀
  5. 将调好的酱汁倒到凉面上, 再加入鸡丝黄瓜丝(还可以加花生米), 搅匀

面食

蒸馒头

  1. 适量酵母，白糖，加300克温水搅匀
  2. 600克面粉加水和面后，揉一会
  3. 醒面
  4. 醒好后揉面，没有气孔后，面团分成几份，揉成长条状，切成四个馒头剂子
  5. 馒头剂子醒20min后，上锅蒸20min即可

肉包

  1. 肉馅：猪肉(带肥肉最好)，大葱，姜末，盐，料酒，生抽，老抽，白胡椒粉，芝麻油，少量水，绞肉机绞碎 1.2 粉丝馅：红薯粉丝开水下锅煮熟后切碎，加入老抽腌制
  2. 和面，蒸和馒头一样

肉饼

肉包包好后，用擀面杖擀至饼状，放入电饼铛煎即可

发面饼

  1. 适量酵母，白糖，加400克温水搅匀
  2. 600克面粉加水，小葱花，盐，和面后，揉一会
  3. 醒面
  4. 醒好后揉面，没有气孔后，面团分成几份，擀成饼状
  5. 醒20min后，上电饼铛烙熟即可

鸡蛋煎饼

小葱末, 鸡蛋, 面粉, 水, 盐, 混合搅匀后上电饼铛煎即可, 3min可煎好一个

肉夹馍

  1. 适量酵母，白糖，加260克温水搅匀
  2. 500克面粉加水，食用油，和面后，揉一会
  3. 醒面
  4. 醒好后揉面，没有气孔后，面团分成几份，擀成饼状
  5. 醒20min后，上电饼铛烙熟即可
  6. 用到切开, 内部涂上烧烤酱, 夹上生菜, 荷包蛋, 煎好的鸡肉或猪肉即可

三明治

  1. 吐司, 烧烤酱
  2. 可选配菜
  • 生菜, 荷包蛋, 烤火腿肠切片, 西红柿切片, 酸黄瓜切片, 烤鸡肉

番茄鸡蛋面

鸡蛋羹

  1. 两个鸡蛋, 250g水, 1/4勺盐, 搅拌均匀
  2. 开水上锅蒸6分钟, 关火后焖5分钟
  3. 出锅后, 加小葱, 香油, 酱油即可

蒸贝贝南瓜

  1. 南瓜洗干净切半, 上锅蒸20min即可
