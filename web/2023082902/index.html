使用Cloudflare + Gmail创建自定义邮件地址, 收发邮件

2023-08-29 19:00
Tags:
  • web
  • dns
  • email

使用Cloudflare的电子邮件路由功能接收邮件

进入站点 -> 电子邮件 -> 电子邮件路由 -> 创建, 设置下自定义邮件地址和转发邮箱地址即可

使用Gmail以自定义邮件地址发送邮件

Gmail -> 设置 -> 全部设置 -> 账号和导入 -> 名字 -> 添加其他的电子邮箱地址

  1. 输入名字和在Cloudflare上创建的自定义邮寄地址, 下一步
  2. 依次输入smtp.gmail.com, 端口号无需修改, gmail邮箱地址@前的部分, gmail密码即google账户密码(如果开启了2FA, 会登陆失败, 这时需要设置一个APP密码, 后述), 点确定没问题即可

设置google app密码

点击头像进入账户 -> 安全 -> 两步验证 -> app密码 -> 生成即可

还是收不到邮件!!

在DNS记录里添加下面值的TXT记录

v=spf1 [IP ADDRESS] include:_spf.google.com include:[DOMAIN] ~all
