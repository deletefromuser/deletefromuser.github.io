用網線連接連接兩台電腦進行文件傳輸

  1. 修改兩台電腦的本地連接設置，ipv4設置到一個網段下
  2. 關閉防火牆
  3. 在高級共享設置中啟用網絡發現
  4. 連接網線
  5. 將要傳輸的文件所在的文件夾設置為共享
  6. done 此時就可以在資源管理器的網絡裡找到對應計算機以及文件進行複製了

ps:找這個主要是為了把steam遊戲搬到新電腦上，比無線局域網快一些，但是比不是移動硬盤的速度（才11m，我的移動硬盤不是固態的，速度都能達到50m）

pps:才11m應該是通過無線局域網的速度，網線可以達到50m速度

才幾天時間，照著上面的都搞了一遍還是不行，找到了通過ip來查看的方法

  1. 輸入\\192.168.3.85在資源管理器中輸入兩個斜桿加目標電腦的ip
  2. 順利的話可以看到該台電腦的共享目錄，點開其中一些可能會發現有些可以點進去查看，有些不可以，提示不可訪問之類的
  3. 這時可以在目標電腦上設置共享目錄的安全屬性，在目錄上右擊菜單在屬性分頁-安全分頁-點擊編輯-添加組或用戶名-在對象名稱中輸入everyone後確定-應用
  4. 此時應該就可以打開該目錄了

通過遠程桌面訪問

  1. 在目標電腦中，控制面板-系統與安全-系統-允許遠程訪問-選擇允許任意版本的遠程桌面連接
  2. 在本機上win鍵+R，輸入mstsc
  3. 輸入目標電腦的ip
  4. 點擊連接，會讓你輸入用戶名密碼，輸入即可
