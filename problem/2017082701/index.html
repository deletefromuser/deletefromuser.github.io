windows修改右鍵“發送到”菜單&如何快速打開SendTo文件夾

右鍵“發送到”所在的文件夾

C:\Users\UserName\AppData\Roaming\Microsoft\Windows\SendTo

如何快速打開

使用shell:sendto

  1. 在文件管理器中輸入上述命令，回車即可
  2. Win鍵+R，在彈出界面中輸入上述命令亦可

ps: shell:startup命令可以快速打開開機啟動文件夾，將軟件快捷方式放到該文件夾下即可實現開機啟動

  2. Shell Commands to Access the Special Folders in Windows 10/8/7/Vista/XP
