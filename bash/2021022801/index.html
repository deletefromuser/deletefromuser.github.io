将所有文件夹的内容分别打包到同名的zip文件内 & 批量解压rar文件

2021-02-28 21:02

下载下来的漫画合集是rar格式，放到perfectview里也不能读，只能解压出来把每一卷压缩成zip文件

以前还写了个sh好像，没想到一行就能搞定

将所有文件夹的内容分别打包到同名的zip文件内

for dir in */; do ( cd "$dir" && zip -r ../"${dir%/}".zip . ) done

批量解压rar文件

for file in ./*.rar; do ( unrar x "$file" ) done
